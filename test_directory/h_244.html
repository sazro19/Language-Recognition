Новые возможности Veeam ONE v11a

Блог компании Veeam Software Системное администрирование *Виртуализация *Резервное копирование *

Как известно, релизы Veeam Backup & Replication и Veeam ONE ходят парой синхронизированы, ибо эти решения входят в Veeam Availability Suite. А посему в скором будущем вместе с новой версией Veeam Backup & Replication выйдет и Veeam ONE v11a. О его новейших фичах и будет мой сегодняшний рассказ. 

Итак, вашему вниманию предлагаются:

  • Поддержка мониторинга и отчетов для облачных бэкапов на Google Cloud Platform.

  • Поддержка immutability для репозиториев.

  • Новые отчеты, включающие информацию о защите инфраструктур под управлением IBM AIX и Oracle Solaris.

Поддержка Veeam Backup for Google Cloud Platform

С помощью Veeam ONE теперь можно мониторить ход резервного копирования для Google Cloud Platform, получать своевременные оповещения и подробные отчеты.

  • Мониторинг и оповещения: помогают следить за тем, чтобы резервное копирование завершалось вовремя и без ошибок, позволяют выявлять политики резервного копирования, которые были деактивированы или выполнялись подозрительно долго.

  • Отчеты: позволяют проанализировать состояние облачной инфраструктуры на предмет соответствия установленному показателю (RPO) и убедиться в защищенности виртуальных машин.

Поддержка репозитория с усиленной защитой - Veeam Backup & Replication Hardened Repository

Возможности мониторинга, оповещений и отчетов теперь доступны и для инфраструктур, в которых используется Veeam Backup & Replication Hardened Repository.

  • Мониторинг и оповещения: упрощают жизнь тем, кто следит за работой вверенных им компонентов инфраструктуры - а именно, должен отслеживать через Data Protection View, настроена ли immutability для стандартных и масштабируемых бэкап-репозиториев (scale-out backup repository, SOBR). Теперь легким движением руки можно настроить автоматическую рассылку оповещений в случае любого изменения настроек immutability.

  • Новые отчеты: дают подробную информацию, в частности, о длительности периода immutability для точек восстановления в Linux Hardened Repository. Позволяют детально изучить настройки immutability для  Linux Hardened Repositories и масштабируемых репозиториев (SOBR), включающих в себя Linux Hardened Repository в качестве экстентов. Помогают отслеживать изменения в настройках immutability.

Мониторинг и отчеты для Veeam Agents for IBM AIX и Oracle Solaris

Для тех, кто работает с IBM AIX и Oracle Solaris, тоже открываются новые возможности по мониторингу и получению отчетов об инфраструктуре. 

Мониторинг и оповещения: с помощью представления Data Protection Views теперь можно следить за тем, как защищены машины с ОС IBM AIX и Oracle Solaris, выявлять те, у которых нет актуальных бэкапов, отслеживать статус политик резервного копирования - в общем, делать все для защиты инфраструктуры.

Отчеты: в новой версии есть набор отчетов, который поможет владельцам инфраструктур на  IBM AIX and Oracle Solaris с обнаружением машин, оказавшихся без резервных копий, с планированием места в репозитории и с другими задачами. 

Поддержка новых платформ

  • Поддерживается мониторинг и предоставление отчетов для инфраструктур, работающих на платформе VMware vSphere 7.0 Update 3.

  • Поддерживаются инфраструктуры на VMware Cloud Director 10.3.

  • Поддерживается Microsoft Windows Server 2022 - как для установки компонентов решения, так и при мониторинге инфраструктур на платформе Microsoft Hyper-V.

Новые RESTful API endpoints для preview

В Veeam ONE 11a для предварительного ознакомления (preview) доступны новые REST API endpoints, которые запланированы к официальному релизу в будущей мажорной версии.  Все желающие смогут ознакомиться с ними и дать свои комментарии и предложения.

В число новых endpoints входят, например:

  • Virtual infrastructure: теперь можно запрашивать данные о компонентах виртуальной инфраструктуры, добавленных в Veeam ONE - от состояния хоста VMware vSphere до отдельной виртуальной машины.

  • Инфраструктура Veeam Backup & Replication и задания (jobs): можно запрашивать данные о заданиях, о компонентах инфраструктуры Veeam Backup & Replication, о метриках резервного копирования и т.д.

  • Сведения о том, защищена ли машина с помощью Veeam Backup & Replication: можно получить и для виртуальных, и для физических, и для облачных инфраструктур.

  • Мониторинг и оповещения: используя RESTful API, можно создавать свои собственные дашборды и настраивать интеграцию с системами ITSM.

  • Business View: можно получить информацию о категориях, объектах и атрибутах Veeam ONE Business View и задействовать ее в ходе интеграции с вашими дашбордами или отчетами. 

Полный список улучшений и обновлений можно найти здесь. 

Установка и апгрейд

Системные требования, список поддерживаемых платформ и known issues вы найдете в Release Notes.  Билд RTM можно скачать отсюда.  

Апгрейд поддерживается для Veeam ONE 9.5 Update 4 (билд 9.5.4.4566) и выше. После апгрейда номер билда станет 11.0.1.1880.

Важно! Если вы апгрейдите полный Veeam Availability Suite, то сначала необходимо забэкапить базу Veeam ONE, как написано тут, затем проапгрейдить Veeam ONE и только после этого приступать к апгрейду Veeam Backup & Replication.
